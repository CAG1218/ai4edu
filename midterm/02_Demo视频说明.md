# Demo 演示说明

## 一、演示视频概述

本演示视频展示了 AI4EDU 智慧教学平台的核心功能与完整使用流程。视频总时长约 5–8 分钟，覆盖从用户注册到核心功能体验的完整路径。

**视频文件命名**：`02_demo_video.mp4`

---

## 二、演示流程说明

### 2.1 启动与访问（0:00–0:30）

1. 启动后端服务：`python -m uvicorn app.main:app --host 0.0.0.0 --port 8000`
2. 启动前端服务：`cd frontend && npx vite --host 0.0.0.0 --port 5173`
3. 打开浏览器访问 `http://localhost:5173/`
4. 页面正常加载，显示 AI4EDU 平台登录界面

**关键截图**：
- 图 1：平台登录页（深蓝渐变背景，AI4EDU 标题）
- 图 2：浏览器开发者工具 → Network 面板，显示 `/api/v1/auth/login` 返回 200

---

### 2.2 用户注册与 Onboarding（0:30–2:00）

1. 点击「注册」切换至注册表单
2. 填写邮箱（如 `demo@ai4edu.com`）、密码、确认密码
3. 点击「注册」按钮，系统提示注册成功并自动跳转 Onboarding 引导页
4. **第一步：选择角色**（学生 / 教师）→ 选择「学生」
5. **第二步：选择兴趣学科**（至少 3 个）→ 勾选「数学」「物理」「计算机」
6. **第三步：选择学习目标** → 勾选「系统学习」「考试提分」
7. 点击「完成引导」，弹出提示「引导完成，开始你的学习之旅！」
8. 自动跳转至平台主界面（不再卡在引导页）

**关键截图**：
- 图 3：Onboarding 角色选择页
- 图 4：Onboarding 兴趣选择页（至少 3 个标签高亮）
- 图 5：Onboarding 目标选择页
- 图 6：引导完成提示 Toast
- 图 7：成功进入主界面（左侧导航栏可见）

> **注**：此前存在 Bug——完成引导后一直卡在引导界面无法跳转。本 Demo 验证了该 Bug 已修复（修复方案：前后端 Onboarding 状态同步）。

---

### 2.3 四大场景模式切换（2:00–3:30）

主界面顶部中央显示场景切换栏，包含四个模式图标：

| 模式 | 图标 | 功能说明 |
|------|------|----------|
| 课堂模式 | 🏫 school | 跟随教师节奏，支持实时问答、投票、举手、分组协作 |
| 自习模式 | 📖 menu_book | 自主学习，AI 导师提供个性化辅导 |
| 考前模式 | 📝 quiz | 模拟考试、错题回顾、计时训练 |
| 讨论模式 | 💬 forum | 小组协作，支持群聊、共享白板、投票 |

**演示操作**：
1. 依次点击四个场景图标，观察页面内容随场景切换动态变化
2. 展示场景切换时 URL 路由同步变化（如 `/classroom`、`/self-study` 等）

**关键截图**：
- 图 8：课堂模式界面
- 图 9：自习模式界面（AI 学伴对话框可见）
- 图 10：考前模式界面
- 图 11：讨论模式界面

### 2.4 AI 学伴对话（3:30–5:00）

在「自习模式」下，右侧显示 AI 学伴对话框：

1. 在对话框中输入：「请解释一下牛顿第二定律」
2. 点击发送，观察 AI 思考动画（...)
3. AI 返回回答（包含公式说明，支持 KaTeX 渲染）
4. 继续追问：「能举个例子吗？」
5. AI 给出具体示例，并保持上下文记忆

**关键截图**：
- 图 12：用户输入问题后的对话框状态
- 图 13：AI 返回的回答（包含公式渲染效果）
- 图 14：多轮对话上下文保持

### 2.5 知识图谱可视化（5:00–6:30）

点击左侧导航「知识图谱」菜单：

1. 页面展示当前学科的知识图谱（基于 D3.js 渲染）
2. 图谱中每个节点代表一个知识点，连线代表知识关联
3. 点击某个节点（如「牛顿运动定律」），右侧显示知识点详情
4. 鼠标悬停节点，高亮关联路径
5. 使用鼠标滚轮缩放图谱，拖拽画布移动视图

**关键截图**：
- 图 15：知识图谱全景（D3.js 力导向图）
- 图 16：点击节点后的详情面板
- 图 17：高亮关联路径效果

### 2.6 智能笔记（6:30–7:30）

点击左侧导航「笔记」菜单：

1. 点击「新建笔记」按钮
2. 在标题栏输入：「牛顿运动定律学习笔记」
3. 在编辑区使用 Markdown 语法编写笔记内容
4. 插入 KaTeX 公式：`$$F = ma$$`
5. 点击「保存」按钮，提示「保存成功」
6. 在笔记列表中看到刚创建的笔记

**关键截图**：
- 图 18：Markdown 编辑器界面（支持实时预览）
- 图 19：KaTeX 公式渲染效果
- 图 20：笔记列表面板

### 2.7 网络状态与离线提示（7:30–8:00）

演示 PWA 离线能力：

1. 在浏览器开发者工具 → Network 面板，勾选「Offline」
2. 页面顶部出现橙色提示条：「离线模式，部分功能受限」
3. 尝试发送 AI 学伴消息，系统提示「已加入离线队列，网络恢复后自动同步」
4. 取消「Offline」，网络恢复
5. 顶部提示条变为蓝色：「同步中...」，随后变为绿色：「同步完成」
6. 离线队列中的消息自动发送

**关键截图**：
- 图 21：离线状态提示（橙色）
- 图 22：离线队列提示
- 图 23：网络恢复后同步完成提示（绿色）

## 三、Demo 环境说明

| 项目 | 说明 |
|------|------|
| 操作系统 | Windows 11 |
| 前端 | Vue 3 + TypeScript，Vite 开发服务器（端口 5173） |
| 后端 | Python 3.13 + FastAPI，Uvicorn（端口 8000） |
| 数据库 | PostgreSQL 17（本地），Neo4j Community 5.x（本地） |
| AI 模型 | OpenAI API（GPT-4o），支持切换其他模型 |
| 浏览器 | Chrome 126+（支持 Service Worker、WebSocket） |

## 四、补充说明

1. **Onboarding 修复验证**：Demo 中重点展示了注册→Onboarding→进入主界面的完整流程，验证了此前「完成引导后卡死」Bug 的修复效果。

2. **局域网访问**：团队成员可通过 `http://[本机IP]:5173/` 访问平台（需在防火墙开放 5173 和 8000 端口）。

3. **AI 回答质量**：Demo 中 AI 学伴的回答质量取决于所接入的模型。当前接入 OpenAI GPT-4o，回答质量较高。后续计划支持本地开源模型部署。

4. **知识图谱数据**：Demo 中的知识图谱为示例数据（高中物理力学模块），正式版本将由教师导入或系统自动构建。
